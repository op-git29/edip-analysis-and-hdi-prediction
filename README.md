📌 Project Title
Economic Development Intelligence & HDI Prediction

📖 Project Description
This project analyzes economic and social data from sources like IMF, World Bank, and UNDP to understand and predict Human Development Index (HDI). The goal is to study how factors like GDP, education, health, and income affect human development.

🎯 Objective
To clean and analyze economic datasets
To identify key factors affecting human development
To build machine learning models for HDI prediction
To compare model performance and extract insights

🛠️ Tools & Technologies Used
Python
Pandas, NumPy
Scikit-learn
Matplotlib, Seaborn
Google Colab
Tensorflow, PyTorch

📊 Workflow
Collected data from IMF, World Bank, and UNDP
Cleaned missing values and handled outliers
Performed exploratory data analysis (EDA)
Built regression and ML models
Evaluated model performance
Visualized insights using charts

📈 Key Insights
GDP and education have strong impact on HDI
Health indicators significantly affect development
Economic stability improves human development index

🚀 Results
The model successfully predicts HDI based on economic indicators and helps understand major drivers of development.
